DSLRRemoteTest.exe is a simple example program to interface with DSLRRemote Pro
and DSLR Remote.
It can be used to run the camera as a simple webcam, adjust the aperture
and shutter speed or simply take individual pictures.

e.g.
First run DSLR Remote or DSLR Remote Pro

To take a single picture:
DSLRRemoteTest

To take a webcam sequence of 5 shots with 45 secs between each shot:
DSLRRemoteTest -w 5 -t 45

Files included:
DSLRRemoteTest.exe - compiled console application
DSLRRemoteLib.dll - DLL used by DSLRRemoteTest.exe to interface with DSLRRemote
DSLRRemoteLib.h - header file for C++ aplpications using the DLL
DSLRRemoteLib.lib - lib for C++ apps to link to the DLL
Source - directory containing a VC++ project and source code for DSLRRemoteTest.exe
