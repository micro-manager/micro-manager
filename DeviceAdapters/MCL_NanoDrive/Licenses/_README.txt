The MCL device adapter static library uses an unmodified version of libusb-1.0.21.  Libusb-1.0.21 and its use are covered by the LGPL version 2.1.  

Included in this folder are:
-A copy of the LGPL license.
-The source code for libusb-1.0.21.
-The object code for the MCL device adapter static library.
