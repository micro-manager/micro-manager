Pre-initialization Properties

  1. This list contains all devices that must be given a particular setting prior to initialization.
  2. More explanation of your devices' pre-initialization properties may be found on individual device pages under the Devices tab on the Micro-Manager website, www.micro-manager.org.
  3. If you are using any serial devices (indicated by the presence of a "Port" in the Property column), press the "Scan Ports" button to see if Micro-Manager can automatically assign ports to these devices.
  4. Some devices do not support automatic port assignment. In that case you will need to tell Micro-Manager which serial port to use to communicate with a particular device.
